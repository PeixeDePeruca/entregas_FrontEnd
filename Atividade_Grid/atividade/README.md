                Atividade de Grid

Esse projeto usando o HTML, CSS e o Grid usando:

- Cabeçalho(header)
- Menu lateral
- Área de conteúdo
- Rodapé(Footer)

Coisas praticadas:

- Layout feito com Grid
- Menu com efeito quando passa o mouse.
- Ícones nos itens do menu
- 2 animações:
  - uma no título (efeito de brilho)
  - outra no conteúdo (aparecendo na tela)
