                      Atividade de Grid

Esse projeto foi feito usando o HTML, CSS e o Grid usando:

- Cabeçalho(header)
- Menu lateral
- Área de conteúdo
- Rodapé(Footer)

Coisas praticadas:

- Layout feito com Grid
- Menu com efeito quando passa o mouse.
- Ícones nos itens do menu
- 2 animações:
  - brilho no título 
  - movimento de slide-in
